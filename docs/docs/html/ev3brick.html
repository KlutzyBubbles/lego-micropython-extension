ev3brick_ext – The EV3 Programmable Brick¶

The ev3brick_ext module is modelled exactly like the core Lego ev3brick module, all methods should work the same when switching although these are some added functionality

Buttons¶

Same as core Lego functions

Light¶

ev3brick_ext.light_flash(color, off_pause=500, on_pause=500)¶

Set the brick light to flash a color

Parameters

  • color (Color) – Color of the light

  • off_pause (int, optional) – Duration of the off pause, defaults to 500

  • on_pause (int, optional) – Duration of the on puase, defaults to 500

ev3brick_ext.light_pulse(color, short_pause=200, long_pause=800, on_pause=100)¶

Set the brick light to pulse a color

Parameters

  • color (ColorExt) – Color of the light

  • short_pause (int, optional) – Duration of the short pause, defaults to 200

  • long_pause (int, optional) – Duration of the long pause, defaults to 800

  • on_pause (int, optional) – Duration of the light flashing on, defaults to 100

Sound¶

classmethod sound.beep(duration=100, volume=30, wait=False)¶

Play a beep/tone

Parameters

  • frequency (int, optional) – Frequency of the beep, defaults to 500

  • duration (int, optional) – Duration of the beep, defaults to 100

  • volume (int, optional) – Volume of the beep, defaults to 30

  • wait (bool, optional) – Whether to wait for the beep to finish before continuing, defaults to False

classmethod sound.beeps(wait=False)¶

Play a number of default beeps with a brief pause in between

Parameters

  • number (int) – Number of beeps

  • wait (bool, optional) – Whether to wait for the beeps to finish before continuing, defaults to False

classmethod sound.file(volume=100, wait=False)¶

Play a sound file

Parameters

  • file_name (SoundFile, str) – Path to the sound file, including extension

  • volume (int, optional) – Volume of the sound, defaults to 100

  • wait (bool, optional) – Whether to wait for the sound to finish before continuing, defaults to False
