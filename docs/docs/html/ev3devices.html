ev3devices_ext – EV3 Motor and Sensor Extension Classes¶

Motor¶

class ev3devices_ext.MotorExt(port, direction=<Direction.CLOCKWISE: 0>, gears=None)¶

Extension class for the Motor device with useful functions mainly relating to speed and specific gear targeting

Further documentation for the Motor class can be found ‘https://klutzybubbles.github.io/lego-micropython-docs/ev3devices.html#ev3devices.Motor’

Parameters

  • port (Port) – Port to which motor is connected

  • direction (Direction, optional) – Direction of the positive motor speed, defaults to Direction.CLOCKWISE

  • gears (list, tuple, optional) – List of gears linked to the motor, defaults to None
