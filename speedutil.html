speed_util.float_percent(percent, min_percent=-100, max_percent=100)¶

Converts a whole percentage (100, 50) to a floating point percentage (1, 0.5).

Adds validation to make sure that the percent is always between min and max.

Parameters

  • percent (percentage whole: %) – The percentage to convert

  • min_percent (percentage whole: %) – Minimum percentage to accept (Default: -100)

  • max_percent (percentage whole: %) – Maximum percentage to accept (Default: 100)

Returns

The floating point percentage between the min and max

Return type

percentage: %

speed_util.get_ratio(gears, depth=None)¶

Gets the ratio of a list or Tuple of gears.

It is reccomended to use a Tuple as genrally gear ratios are not going to be changed dynamically.

This method should NOT be used as a replacement for the inbuilt motor gear ratios.

Parameters

  • gears (list, tuple) – List of gears from input to output, gear trains can be used by nesting lists or tuples inside

  • depth (int) – Depth of train gears to get the ratio for, works like array pointers or you can set it to None to ignore it (Default: None)

Returns

Gear ratio floating point

Return type

float
